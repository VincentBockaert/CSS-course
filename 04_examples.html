Classic page layout

Steps

  • To layout a page, you usually perform these steps:
    1. set the width and/or height of the larger content blocks
      and layout them, mostly with float
    2. layout all content within each block, with positioning and/or float
    3. apply other styles like backgrounds, fonts, margins...
  • → Let's try this out with a realistic example
  • → We'll take a page from here to there.

Center the wrapper

  • We start with step 1: set the width and/or height of each major block, and float or position it.
  • The wrapper: 
  • #wrapper {
  width: 900px;
  margin: 0 auto;
}
  • We leave the header where it is (on top), and just set the height: 
  • #wrapper > header {
  height: 60px;
}

Create horizontal menu

  • The main menu we also leave where it is (just below the header), and just layout the links horizontally with inline-block: 
  • #mainmenu ul li {
  display: inline-block;
}

Create two columns

  • The submenu and main content columns in the wrapper are placed next to each other by setting the width and floating left: 
  • #submenu {
  width: 180px;
  float: left;
}

#content {
  width: 640px;
  float: left;
}

Clear the footer

  • There's no need to restore the wrapper with clearfix; clearing the footer will do the trick: 
  • #wrapper > footer {
  clear: both;
}
  • Finally we place the footer links next to each other with inline-block: 
  • #wrapper > footer li {
  display: inline-block;
}

Layout within blocks

  • Up to step 2: float or position the content within each block
  • To place <h1> and the login <p> in the header, we'll use relative-absolute positioning: 
  • #wrapper > header {
  /*...*/
  position: relative;
}

#wrapper > header h1 {
  position: absolute;
  top: 10px;
  left: 20px;
  margin: 0;
  padding: 0;
}

#wrapper > header p {
  position: absolute;
  top: 10px;
  right: 20px;
  margin: 0;
  text-align: right;
}

Float aside right

  • The <aside> in the main content is simply floated right with a border: 
  • #content aside {
  float: right;
  width: 130px;
  border: 1px solid gray;
}

Style main menu

  • The main menu links layout: 
  • #mainmenu ul li {
  display: inline-block;
}
#mainmenu ul li a {
  display: inline-block; /* or width and height won't work */
  line-height: 24px; /* height - 2 * padding */
  height: 28px;
  width: 50px;
  padding: 2px 5px;
  text-align: center;
}
#mainmenu ul li + li:before {
  content: '|';
  color: #999;
}
  • Note how we:
    • set inline-block on <a> too
    • set width and height of <a> instead of <li> to maximize click area
    • align vertically with line-height
    • use li + li to select any list item except the first
    • use :before to draw the vertical separators

Note how we wrap checkboxes and radiobuttons in their labels, while the other controls are placed outside their labels. Both syntaxes are allowed, but this combination turns out to be the easiest to layout with CSS.

Steps

  • Our plan is quite simple:
    • set the width of <form> and <dl> to 660px
    • set the width of <dt> to 150px and <dd> to 500px
    • float all <dt> and <dd> left, and
    • restore <dl> with clearfix
    • place radiobuttons and checkboxes vertical by setting <label> blocklevel
    • vertically align labels and controls in <dt> and <dd>
    • finish with some margins, paddings, fonts etc...

    CSS sprites (4)

    • CSS sprites are used by large sites like Youtube and Amazon
